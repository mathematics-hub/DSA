# [Find triplets with zero sum](https://www.geeksforgeeks.org/problems/find-triplets-with-zero-sum/1?page=3&category=Arrays&sortBy=submissions)
## Medium
Given an array&nbsp;arr[] of integers, determine whether it contains a triplet whose sum equals zero. Return true&nbsp;if such a triplet exists, otherwise, return false.
Examples:
Input: arr[] = [0, -1, 2, -3, 1]Output: true
Explanation: The triplet [0, -1, 1] has a sum equal to zero.
Input: arr[] = [1, 2, 3]Output: false
Explanation: No triplet with a sum of zero exists.
Input: arr[] = [-5, 3, 2, -1, 0, 1]Output: true
Explanation: The triplet [-5, 3, 2] has a sum equal to zero.
Constraints:1 ≤ arr.size() ≤ 103-106 ≤ arr[i] ≤ 106
