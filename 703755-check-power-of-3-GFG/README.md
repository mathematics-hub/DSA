# [Check Power of 3](https://www.geeksforgeeks.org/problems/check-a-integer-is-power-of-3-or-not3850/1)
## Easy
Given a positive integer n, find if it is a power of 3.&nbsp;A number is said to be a power of 3 if it can be expressed as 3x&nbsp;for some non-negative integer x.
Examples:
Input: n = 3
Output: true
Explanation: 3 = 31, so it is a power of 3.
Input: n = 1
Output: true
Explanation: 1 = 30, so it is a power of 3.
Constraints:1 ≤ n ≤ 109
